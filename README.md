# Python Network Broadcast Load Generator (Educational)

This project provides a Python-based tool for generating controlled UDP broadcast traffic on a local network.  
It is intended for **educational purposes**, network behavior analysis, and controlled lab testing.

⚠️ Important Warning  
Use this tool **only** in your own network, test labs, or controlled environments.  
Improper use can disrupt network operations.  
You are fully responsible for using this tool safely and legally.

---

## Features
- Generates UDP broadcast traffic at a target bandwidth
- Multi-process load generation
- Precise packet timing using nanosecond scheduling
- Useful for learning about networking, bandwidth, and system behavior
